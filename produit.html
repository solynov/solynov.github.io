SOLYNOV Factur-X

Un logiciel Windows conçu pour produire des factures électroniques compatibles Factur-X sans changer toute votre organisation.

Le principe

Vous partez de votre facture PDF habituelle. SOLYNOV ajoute les données structurées nécessaires pour générer une facture électronique au format compatible Factur-X.

Le bénéfice

Vous continuez à travailler simplement, tout en produisant des fichiers adaptés aux nouvelles obligations de facturation électronique.

Fonctionnalités prévues

Import PDF

Utilisez votre facture PDF comme base de travail.

Données depuis tableur

Alimentation possible via une organisation simple et compréhensible.

Génération Factur-X

Création d’un PDF enrichi avec les données électroniques attendues.

Contrôle conformité

Factures conçues pour être vérifiées avec des validateurs classiques comme Mustang ou d’autres outils spécialisés.

Mises à jour

Prévu pour évoluer si les règles ou besoins réglementaires changent.

Licence

Page prête à recevoir le système d’achat, clé d’activation et gestion de licence.

Positionnement clair

SOLYNOV n’est pas un ERP. C’est une solution simple pour les petites entreprises qui veulent produire des factures électroniques sans complexité.
